# 接口文档

## 病人部分

### 登录

类型：`post`

路径：`api/login/`

输入：

```json
{
    'username' :string,
    'password' :string
}
```

输出：

```json
{
    'code': 0, 
    'message': string, 
    'type': string,
    'data':
    	{
            'token': string,
            'expires': double
        }
}
```

### 注册

类型：`post`

路径：`api/signUp/`

输入：

```json
{
    'username' : string, 
    'password' : string,
    'idcard' : string,
}
```

输出：

```json
{
    'code' : Number
}
```

### 诊室信息

类型：`post`

路径：`api/departmentInfo/`

```json

}
```

```json
{
    'doctorList':[
        {
            'name' : string,
            'room' : string,

```
